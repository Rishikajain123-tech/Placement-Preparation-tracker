# 🚀 Placement Preparation Tracker

A responsive web application built using **HTML, CSS, and Vanilla JavaScript** to help students organize and track their placement preparation efficiently. The application allows users to manage company-wise DSA questions, monitor progress, and store data locally for future access.

## ✨ Features

* ✅ Add new DSA questions
* ✏️ Edit existing questions
* 🗑️ Delete questions
* 💾 Local Storage support (data persists after refresh)
* 🔍 Live Search
* 🏢 Filter by Company
* 📚 Filter by Topic
* 📊 Dashboard Statistics

  * Total Questions
  * Solved Questions
  * Remaining Questions
  * Progress Percentage
* 🚫 Duplicate Question Prevention
* 📱 Responsive Design

## 🚀 How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/Rishikajain123-tech/Placement-Preparation-tracker.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

No additional installation or dependencies are required.

---

## 📖 How It Works

* Add a coding question with company, topic, difficulty, status, and LeetCode link.
* Questions are stored in the browser using Local Storage.
* Search questions instantly by title, company, or topic.
* Filter questions based on company and topic.
* Track preparation progress through the dashboard.
* Edit or delete questions whenever required.
